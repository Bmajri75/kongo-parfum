# 🛍️ Kongo Parfum - E-Commerce Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Node.js](https://img.shields.io/badge/Node.js-v18+-green)](https://nodejs.org/)
[![Angular](https://img.shields.io/badge/Angular-21-red)](https://angular.io/)

Une plateforme e-commerce complète pour la vente de parfums haut de gamme, conçue avec les dernières technologies web.

## ✨ Fonctionnalités principales

- 🛒 **Panier d'achat dynamique** - Gestion fluide du panier avec sync en temps réel
- 💳 **Paiement sécurisé** - Intégration des passerelles de paiement
- 👤 **Gestion des utilisateurs** - Profils, historique d'achats, favoris
- 📦 **Gestion d'inventaire** - Suivi des stocks en temps réel
- 🔍 **Recherche & filtrage** - Filtres avancés par catégorie, prix, notes
- 📱 **Design responsive** - Optimisé pour desktop, tablet, mobile
- 🎨 **Interface moderne** - UX/UI professionnelle et intuitive

## 🏗️ Stack Technique

### Frontend
- **Angular 21** - Framework frontend réactif
- **TypeScript** - Typage strict pour fiabilité
- **RxJS** - Programmation réactive
- **SCSS** - Styling avancé

### Backend
- **Node.js** - Runtime JavaScript
- **Base de données** - Gestion produits et commandes

## 🚀 Installation & Démarrage

### Prérequis
- Node.js v18+
- npm v11+

### Frontend
```bash
cd kongo-frontend
npm install
npm start
# L'app démarre sur http://localhost:4200
```

### Backend
```bash
cd kongo-backend
npm install
npm start
```

## 📋 Commandes principales

```bash
# Développement
npm start

# Build production
npm run build

# Tests unitaires
npm test

# Linting
npm run lint
```
